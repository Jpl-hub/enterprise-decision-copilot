# 数据治理与合规原则

## 一、正式数据源范围

正式入库数据优先来自以下赛题给定来源：

1. 上交所定期报告
2. 深交所定期报告
3. 北交所公告页
4. 东方财富个股研报与行业研报
5. 国家统计局公开统计数据

## 二、合规边界

- 只采集公开披露、公开展示的数据。
- 不绕过登录、鉴权、验证码或访问控制。
- 对国家站点与交易所站点坚持低频、定向、必要访问。
- 宏观统计优先采用人工核验后入库，不做高频自动扫站。
- 每条核心数据都保留 `source_url` 与处理路径。

## 三、数据分层

- Bronze：原始抓取结果、原始 PDF、原始 JSON、原始 HTML
- Silver：清洗去重后的结构化表
- Gold：面向评分、问答、报告生成的主题宽表

## 四、质量校验

- 唯一性：公司代码 + 报告年份唯一
- 完整性：关键字段为空比例监控
- 一致性：代码、简称、市场、年份格式统一
- 可追溯性：保留来源链接、抓取时间、处理脚本
- 异常值：收入、利润、现金流、ROE 等关键字段做范围检查
